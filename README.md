# Books Library — Навчальна практика 📚

Індивідуальний full-stack веб-застосунок для керування каталогом книг. Проєкт розроблено в межах навчальної практики з дисципліни «Сучасні технології Internet-програмування».

## 🛠 Технологічний стек
* **Backend:** Node.js, Nest.js, TypeScript (In-memory storage)
* **Frontend:** React (Vite), TypeScript, React Router
* **Інструменти:** Git, GitHub, VS Code

## 📂 Структура проєкту
Проєкт реалізований у форматі monorepo:
- `backend/` — REST API сервер на Nest.js.
- `frontend/` — клієнтська частина на React.

## 🚀 Функціональні можливості (у розробці)
- [ ] Перегляд списку книг та авторів.
- [ ] Додавання нових книг до бібліотеки.
- [ ] Редагування інформації про книги.
- [ ] Видалення книг з каталогу.
- [ ] Валідація вхідних даних.
